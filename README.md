# FoxbodyDash

A modern digital dashboard designed specifically for the 1988 Ford Mustang Foxbody.

FoxbodyDash is a touchscreen-based instrument cluster that runs on a Raspberry Pi and communicates with the FoxbodyBCM and engine management system to provide a fully customizable OEM-quality digital driving experience.

---

## Features

### Current
- SVG analog tachometer
- SVG analog speedometer
- Needle sweep animation
- Digital RPM display
- Digital MPH display
- Touchscreen interface
- Modular JavaScript architecture
- Responsive dashboard layout

### Planned
- Live ECU data
- Coolant temperature gauge
- Oil pressure gauge
- Fuel level gauge
- Battery voltage gauge
- Startup animation
- Warning light icons
- Shift lights
- Day/Night themes
- Multiple dashboard layouts
- Data logging
- Diagnostics page
- Vehicle settings
- GPS integration
- Music controls

---

## Hardware

Current target hardware:

- Raspberry Pi 4
- 10.1" Capacitive Touch Display
- HDMI LCD
- CAN/LIN expansion (future)
- FoxbodyBCM integration
- MicroSquirt ECU
